Improving Accuracy by Using Multiple Linear Regression

Chandler Rottenberg

     A simple linear regression model uses one variable to attempt to predict an outcome. While this may offer enough accuracy to be considered right most of the time, real world problems will usually have multiple variables that affect what happens. Ignoring these extra variables can cause problems if we want to rely on our model's predictions. How can we create a model that includes these variables, and does the extra work actually improve precision enough for it to be worth it?
